# Projeto Validação MVC

Projeto desenvolvido para demonstrar validação de formulário utilizando JavaScript.

## Requisitos implementados

- document.getElementById()
- addEventListener("submit")
- event.preventDefault()
- Validação de nome
- Validação de email
- Validação de idAde
- Exibição de mensagens com textContent
